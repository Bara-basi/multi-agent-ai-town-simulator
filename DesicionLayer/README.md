# AITown Decision Layer

## 项目背景

AITown 是一个以“AI 小镇”为题材的多智能体仿真项目。这个系统并不把LLM当作一次性问答工具，而是把它放进一个持续运行的世界中：多个角色拥有各自的状态、记忆、目标与资源约束，会在统一的环境规则下不断观察局势、制定计划、执行动作、处理失败并进行反思，形成可持续运转的行为闭环。

这个项目想解决的核心问题是：如何把大模型从“会对话”，推进到“能在复杂环境中长期行动”。在 AITown 中，Agent 不只是输出文本，而是要基于世界状态做出真实决策，例如移动、交易、消耗物品、休息恢复，以及在资源不足、行动受阻或策略失效时重新规划。为了让这些行为具备稳定性，系统在架构上补充了状态管理、动作校验、错误恢复、重复行为保护、交易约束和阶段性反思等机制，使 Agent 的行为不仅“像在思考”，也“真的会受到环境约束”。

从作品集视角看，AITown 更接近一个完整的 Agent 系统原型，而不是单点功能 Demo。它同时覆盖了多智能体调度、Prompt 驱动决策、世界建模、经济系统、运行时保护、可视化监控，以及 Python 决策层与 Unity 执行层的联动。这个项目体现的重点不是某一个模型调用技巧，而是如何围绕大模型搭建一套可运行、可扩展、可调试的智能体工程框架，并将其落到一个具备交互感和观察价值的“小型社会模拟”场景中。
